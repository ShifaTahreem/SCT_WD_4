# TaskFlow - To-Do Web Application

## Overview

This project is a simple and user-friendly To-Do Web Application developed as part of the SkillCraft Technology Web Development Internship (Task 04).

The application allows users to create, manage, and organize daily tasks. Users can add tasks with date and time, mark tasks as completed, edit existing tasks, delete tasks, and maintain their task list efficiently.

## Features

* Add New Tasks
* Edit Existing Tasks
* Delete Tasks
* Mark Tasks as Completed
* Set Date for Tasks
* Set Time for Tasks
* Task List Organization
* Task Counter (Total and Completed Tasks)
* Data Persistence using LocalStorage
* Dark / Light Theme Toggle
* Responsive Design for Mobile and Desktop
* Modern User-Friendly Interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## How to Use

1. Enter a task in the input field.
2. Select the required date and time.
3. Click **Add Task** to create a task.
4. Click **✔** to mark a task as completed.
5. Click **✏** to edit a task.
6. Click **✖** to delete a task.
7. Use the theme toggle button to switch between dark and light modes.
